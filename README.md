# Mood Diary — Цифровой дневник настроений

Визуальное приложение для отслеживания эмоционального состояния с тёмной кинематографичной темой и неоновыми акцентами.

## Возможности

- Выбор из 6 эмоций — при выборе вся цветовая схема страницы плавно меняется
- Heatmap-календарь настроений (круглые ячейки с градиентами)
- CSS-график статистики по эмоциям за месяц
- Лента последних записей с возможностью удаления
- Все данные хранятся в localStorage
- 20+ демо-записей при первом запуске
- Полная адаптация под мобильные устройства

## Технологии

- React 18 + TypeScript
- Framer Motion — анимации появления и удаления
- CSS — кастомные стили, переменные, адаптивная вёрстка
- localStorage — хранение данных

## Дизайн

- Тёмная тема: фон `#0a0a0a`
- Акценты: фиолетовый `#8b5cf6`, янтарь `#f59e0b`, розовый `#ec4899`
- Шрифты: Instrument Serif (заголовки) + Space Grotesk (текст)

## Запуск

```bash
npm install
npm start
```

## Сборка

```bash
npm run build
```
