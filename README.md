# Mood Diary — Цифровой дневник настроений

Визуальное приложение для отслеживания эмоционального состояния с красивой тёмной темой и неоновыми акцентами.

![Screenshot](./screenshot.png)

## Возможности

- **Дэшборд** с heatmap-календарём настроений, D3.js графиком за 2 недели и статистикой
- **Новая запись** — выбор из 5 эмоций с анимированными иконками и заметкой
- **История** — полный список записей с поиском, фильтрацией и удалением
- Все данные хранятся в localStorage
- Адаптивная вёрстка для мобильных устройств
- 18 демо-записей при первом запуске

## Технологии

- **React 18** + **TypeScript**
- **D3.js** — графики с плавными кривыми и градиентной заливкой
- **Framer Motion** — анимации переходов, списков, модалок
- **date-fns** — работа с датами
- **CSS** — кастомные стили без фреймворков

## Дизайн

- Тёмная тема: фон `#0a0a12`
- Неоновые акценты: фиолетовый `#7c3aed`, голубой `#06b6d4`
- Шрифт: Space Grotesk

## Запуск

```bash
npm start
```
